# Sagittarum - AI Grading Platform

Static HTML interface for [https://virtualvoyager367.github.io/Sagittarum/](https://virtualvoyager367.github.io/Sagittarum/).

## Structure

```
Sagittarum/
├── index.html          → Company overview (/)
├── signin/
│   └── index.html      → Sign in (/signin/)
├── signup/
│   └── index.html      → Sign up (/signup/)
├── student/
│   └── index.html      → Student dashboard (/student/)
└── teacher/
    └── index.html      → Teacher dashboard (/teacher/)
```

Each HTML file is self-contained with inline CSS and JavaScript. No build step or dependencies.

## GitHub Pages

Enable Pages in repository Settings → Pages → Publish from branch (root).
